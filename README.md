
# 3-Joint Robotic Arm

BlueStamp Engineering student portfolio

---

## Project Description
For my BlueStamp Engineering summer project, I built and programmed a 3-Joint Robotic Arm that uses an Arduino microcontroller and servo motors to replicate basic human joint movements. My main goal was getting the three independent axes to work together smoothly so the arm could accurately grab and move objects. The biggest challenge was dealing with mechanical wobble and finding the right balance in the code so the servos didn't strain or bind up during tight rotations. After structural fine-tuning and debugging my code, I stabilized the arm and got it running smoothly. This project gave me awesome hands-on experience with hardware troubleshooting, mechanical assembly, and C++ programming.
